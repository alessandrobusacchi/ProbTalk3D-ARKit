This folder will contain the processed dataset. 
- param folder will contain the animation data in FLAME parameters space
- wav folder will contain the audio files
- prosody_static will contain extracted prosody features of the audio files
- use prosody_extract.py to extract the features 
- [DisVoice](https://github.com/jcvasquezc/DisVoice/tree/master/disvoice/prosody) is needed to run the `prosody_extract.py` script
